**npm start: roda o projeto
*npx create-react-app nome-projeto: (cria projeto em react)


ctrl+c: para de rodar o projeto


*componentes de classe se usa o render
*componentes funcionais n usam render
*pode se exportar a classe no final ou direto integrado a classe (export default login;/ export default class...)
*Campos de classe (buscarEventos = () =>{})
*fetch: faz a requisiçaõ através da URL passada dentro dela
*.then(): se refere ao retorno de uma requisição
*catch: tratativa de erro
--thead: faz o cabeçalho da "tabela" 
*setItem: grava um valor
*getItem: pega um valor




----------OBS--------
*não é possível fazer alterações sem ter funções criadas;
* na criação do projeto o nome tem que ser em minúsculo (npx create-react-app nome-projeto);
*quando se clona faz :npm install (para baixar as node modules)
